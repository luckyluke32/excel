# Excel

This repository contains a script to generate an Excel file with restaurant names and emails for every municipality in Attica, Greece.

## Usage

1. Obtain a Foursquare API key and set it in the environment variable `FOURSQUARE_API_KEY`.
2. Run the script:
   ```bash
   python generate_attica_restaurants.py
   ```
3. The script will create `attica_restaurants.xlsx` in the current directory.
